== Description ==

Miscellany is a wordpress plugin which is used to build portfolio.

Miscellany plugin create a custom post type. Admin can create many portfolio of custom post type miscellany.

== Installation ==

1. Download the plugin from WordPress repository.
2. Go to WordPress admin panel > Plugins > Add New.
3. Upload `miscellany.zip` to the by clicking on upload button.
4. Activate the plugin on the next screen.
